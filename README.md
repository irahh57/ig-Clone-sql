## ig-Clone-sql

### About the Project
A relational database project that replicates core Instagram features 
using SQL — including users, posts, likes, comments, and followers.

### Tools Used
- SQL (MySQL)

### What this project covers
- Database schema design
- Table relationships and foreign keys
- Queries for user activity, popular posts, and engagement analysis

### Files
- `ig_clone.sql` — Full database schema and queries
- `Instagram_Analytics_Report.pptx` — Analytics report
